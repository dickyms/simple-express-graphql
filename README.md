# Simple Express and GraphQL run on Docker

### Installation

1. Clone Repository
```sh
   git clone https://github.com/dickyms/simple-express-graphql.git
   ```
2. Install package
```sh
   npm install
   ```
3. Run project
```sh
   node .
   ```
